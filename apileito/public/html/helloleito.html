API de Gestión de Items y Usuarios

Sistema para administrar items de juego y usuarios

Documentación de la API

Método Endpoint Descripción Parámetros
GET /api/items Obtiene todos los items del catálogo -
GET /api/items/:id Obtiene un item específico por ID id - ID del item
POST /api/items Crea un nuevo item nombre, tipo, efecto
PUT /api/items/:id Actualiza un item existente id - ID del item, nombre?, tipo?, efecto?
DELETE /api/items/:id Elimina un item id - ID del item
GET /api/usuarios Obtiene todos los usuarios con sus items -
GET /api/usuarios/:id Obtiene un usuario específico por ID con sus items id - ID del usuario
POST /api/usuarios Crea un nuevo usuario nombre, correo, items?
PUT /api/usuarios/:id Actualiza un usuario existente id - ID del usuario, nombre?, correo?, items?
DELETE /api/usuarios/:id Elimina un usuario id - ID del usuario
