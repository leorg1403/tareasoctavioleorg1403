CSS Basicos

CSS cascade style sheet es la forma en la que se puede dar estilo estatico a una pagina web

CSS se puede aplicar de tres formas diferentes:

  • CSS en linea: Se aplica directamente en el elemento HTML.

    Ejemplo:

    <p style="color: blue; font-size: 16px;"> Este es un parrafo con estilo en linea</p>

    Este es un parrafo con estilo en linea

  • CSS interno: Se aplica dentro de la etiqueta <style> en la sección <head> del documento HTML.

    Ejemplo:

    <head>
    <style>
    p {
        color: green;
        font-size: 18px;
    }
    </style>
</head>
  • CSS externo: Se aplica a través de un archivo CSS separado que se vincula al documento HTML.

    Ejemplo:

    <head>
    <link rel="stylesheet" href="styles.css">
</head>

Selectores CSS

Los selectores determinan a que elemento se le aplicara un estilo

Ejemplo de los selectores

/* Selector tipo*/
        p{color: blue;}

/* Selector de clase*/
.mi-clase{font-size: 20px;} 

/* Selector de id*/
#mi-id {font-weight: bold;}

/* Selector descendiente*/
article p{line-height. 1.5;}

/* Selector hijo directo*/
ul > li{list-style-type: square;}

/* Selector de atributo*/
input[type="text"]{border: 1px solid black;}

Modelo de Caja (Box Model)

Todo elemento HTML es una caja rectangular compuesta de contenido, padding, borde y margen:

Ejemplo de Box Model:

.box {
width: 200px;         /* Ancho del contenido */
padding: 20px;        /* Espacio interno */
border: 5px solid green;   /* Borde */
margin: 20px;         /* Espacio externo */
background-color: yellow;
}

↑ Esta caja tiene: width 200px, padding 20px, borde 5px, y margen 20px

Flexbox (Layout Unidimensional)

Flexbox es un modo de diseño que permite distribuir elementos en una dirección (fila o columna):

Ejemplo de Flexbox:

.contenedor {
display: flex;
justify-content: space-between;
}

Transiciones y Animaciones

Las transiciones permiten cambiar suavemente de un estilo a otro:

Ejemplo de transición (pasa el cursor por encima):

.elemento {
transition: transform 0.3s ease;
}
.elemento:hover {
transform: scale(1.1);
}
Hover me

Ejemplo de animación:

@keyframes slide-in {
from { transform: translateX(-100%); }
to { transform: translateX(0); }
}
.elemento {
animation: slide-in 2s ease;
}

Variables CSS (Custom Properties)

Las variables CSS permiten reutilizar valores en todo el código:

Ejemplo:

:root {
--color-primario: #007bff;
--espaciado: 20px;
}

.elemento {
color: var(--color-primario);
margin: var(--espaciado);
}
