Comandos básicos de Bash

Comando Descripción
pwd Muestra la ruta actual en la que estás ubicado
ls Lista los archivos y carpetas del directorio actual
cd Cambia al directorio especificado
clear Limpia la pantalla de la terminal
history Muestra los comandos utilizados recientemente
cat Muestra el contenido de un archivo
mkdir Crea una nueva carpeta
touch Crea un archivo vacío
nano / vim Editores de texto para terminal
rm Elimina un archivo
cp Copia archivos o directorios
mv Mueve o renombra archivos
grep Busca patrones en archivos
find Busca archivos en el sistema
chmod Cambia permisos de archivos
sudo Ejecuta comandos con privilegios de administrador
ps Muestra los procesos en ejecución
kill Termina un proceso
